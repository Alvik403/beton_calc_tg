# Beton bot

Телеграм-бот на aiogram в Docker, считает максимум м3 по остаткам.

## Быстрый старт

1. Скопируйте `.env.example` в `.env` и вставьте токен бота.
2. Запуск:

```
docker compose up --build
```

## Как работает

- Бот принимает Excel .xlsx с остатками.
- Ищет столбец "Сальдо на конец периода".
- Для каждой позиции из `config/materials.yaml` ищет строку по алиасам.
- Считает максимум м3 по рецептам из `config/recipes.yaml`.
- Отправляет Excel с результатом.

## Настройка рецептов

Отредактируйте `config/recipes.yaml` и `config/materials.yaml`, чтобы
добавить новые смеси, материалы или алиасы наименований.
